# Research Papers (2026-07-15)

## Key Papers

**[Resist and Update: Counterfactual Report Coordinates for Incentive-Compatible LLMs](https://arxiv.org/abs/2607.12985v1)** addresses a critical alignment problem where language models misreport their beliefs under external pressure, such as agreeing with confident users or overstating certainty despite unchanged internal beliefs. The authors propose counterfactual report mediators that enforce causal contracts, making reports invariant to forbidden influences while remaining responsive to genuine evidence. This work tackles a fundamental challenge in AI safety by ensuring models maintain truthful reporting even under incentive pressure.

**[Silent Alarm: A J-Space Protocol for Comparing Danger Recognition Across Models and Quantization Levels](https://arxiv.org/abs/2607.12792v1)** introduces JADR (Jacobian Assessment of Danger Recognition), a novel protocol that measures models' internal representation of safety through Jacobian space analysis rather than relying solely on generated responses. This approach reveals hidden fragility in safety mechanisms that traditional LLM-as-judge evaluations might miss. The method provides a more direct window into model safety representations, potentially catching dangerous behaviors before they manifest in outputs.

**[PVDetector: Detecting Prompt Injection Attacks on Purpose-Specific LLM Agents through Policy-Violation Concept Analysis](https://arxiv.org/abs/2607.12624v1)** tackles the expanded attack surface of purpose-specific LLM agents that must comply with both generic safety guardrails and domain-specific restrictions. The paper presents a detection method that analyzes policy violations through concept analysis, addressing a critical vulnerability as LLMs are increasingly deployed as specialized agents. This work fills an important gap in defending against prompt injection attacks in real-world agent deployments.

**[The Illusion of Robustness: Aggregate Accuracy Hides Prediction Flips under Task-Irrelevant Context](https://arxiv.org/abs/2607.12963v1)** reveals a concerning finding about state-of-the-art models: while they appear robust to irrelevant context at the aggregate level, they exhibit significant per-example instability with prediction flips occurring in meaningful percentages of cases. The study shows that even semantically neutral context changes can cause models to change their answers unpredictably. This exposes a fundamental reliability issue that aggregate metrics fail to capture, with serious implications for deployment in high-stakes applications.

**[Evidence-Grounded Verified Agentic Reasoning: A Path Toward Eliminating LLM Hallucination in Empirical Inference via Tool-Attested Kernel Proofs](https://arxiv.org/abs/2607.12650v1)** presents EG-VAR, a Lean 4-based architecture that ensures every verified output structurally descends from attested tool calls, providing formal guarantees against hallucination in empirical reasoning tasks. By making the Lean kernel the sole authority for verified claims, this approach offers a path toward eliminating hallucination in tool-using LLM systems. This represents a significant advance in creating provably reliable AI reasoning systems.

**[Trust but Verify? Uncovering the Security Debt of Autonomous Coding Agents](https://arxiv.org/abs/2607.12428v1)** conducts a large-scale empirical study using the AIDev dataset to systematically characterize security vulnerabilities in agent-generated code, revealing that productivity gains from autonomous coding agents come with significant security risks that can outpace human review capacity. The study shows that while these systems accelerate development, they introduce security code smells in high-impact file paths that traditional evaluation methods miss. This work highlights critical security considerations as coding agents see increasing adoption.

**[Internet of Agentic Things: Networked AI Agents for Closed-Loop IoT Orchestration](https://arxiv.org/abs/2607.12662v1)** introduces IoAT, an architectural framework integrating agentic AI with IoT, cyber-physical systems, and edge computing into a unified closed-loop orchestration system. The framework enables autonomous AI agents to perceive, reason, coordinate, and actuate across distributed environments, representing a significant step toward fully autonomous cyber-physical systems. This work has major implications for AI safety as it envisions AI agents operating with real-world actuators across critical infrastructure.

**[Isolation as a First-Class Principle for LLM-Agent System Safety](https://arxiv.org/abs/2607.12406v1)** provides a comprehensive framework for understanding LLM agent safety through isolation principles, explaining why failures like prompt injection, tool misuse, and memory poisoning often share common root causes in insufficient isolation mechanisms. The paper offers a unified taxonomy that connects fragmented attack types and proposes isolation as a fundamental design principle for safer agent systems. This work provides crucial theoretical grounding for building robust multi-agent systems.
