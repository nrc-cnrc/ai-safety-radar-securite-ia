# Daily Digest (2026-05-13)

## Top 3 Developments

**Frontier AI Monitoring Faces Critical Scalability Challenge as Models Fail to Detect Dangers in Long Contexts**: [Research reveals](https://arxiv.org/abs/2605.12366v1) that GPT, Claude, and Gemini increasingly fail to detect dangerous actions in lengthy coding agent transcripts, with performance degrading significantly beyond 100K tokens despite real-world monitoring requiring 500K+ token contexts. This matters because it exposes a fundamental limitation in our ability to monitor AI agents as they become more capable and generate longer interaction histories.

**MIRI Proposes International Superintelligence Moratorium as Concrete Governance Framework**: [The Machine Intelligence Research Institute has published](https://intelligence.org/2026/05/12/summary-an-international-agreement-to-prevent-the-premature-creation-of-artificial-superintelligence/) a detailed summary of their proposal for an international agreement to prevent premature artificial superintelligence development, arguing current methods would likely lead to catastrophic outcomes. This matters because it represents one of the most concrete governance proposals from a leading AI safety organization for coordinated global action on existential risk.

**New Attack Surface Discovered in Agent Skill Ecosystems Through Malicious Capability Modules**: [Research demonstrates](https://arxiv.org/abs/2605.12015v1) that reusable skill modules can compromise agent safety even when user requests are benign, as malicious content in skill materials or local artifacts can steer agents toward unsafe actions through previously overlooked attack vectors. This matters because it reveals how the growing ecosystem of modular AI capabilities creates new security vulnerabilities that traditional safety evaluations may miss.

## Risk Watch

**Increased Risks**

**AI safety monitoring systems fail to detect dangerous actions in realistic deployment contexts**, as demonstrated by [research showing](https://arxiv.org/abs/2605.12366v1) frontier models' performance degrading beyond 100K tokens while real-world monitoring requires 500K+ token contexts. **Modular agent skills create new attack vectors that bypass traditional safety evaluations**, as revealed in [research demonstrating](https://arxiv.org/abs/2605.12015v1) how malicious content in reusable skill modules can compromise agent safety even with benign user requests. **Knowledge graph-enhanced AI systems face novel backdoor attack vulnerabilities**, as shown in [research exposing](https://arxiv.org/abs/2605.11996v1) how adversaries can manipulate retrieved subgraphs to trigger malicious behaviors while bypassing text-based defenses. **Tool input validation gaps in AI systems create security vulnerabilities**, highlighted by [community discussions](https://github.com/anthropics/claude-cookbooks/issues/612) around how untrusted content can manipulate tool behavior through input shaping.

**Decreased Risks**

**Surgical neuron-level safety interventions enable precise control without degrading model coherence**, as achieved through [research identifying](https://arxiv.org/abs/2605.12290v1) the 0.1% of neurons most responsible for distinguishing harmful from benign prompts. **Token-level hallucination detection provides finer granularity and better scalability for error identification**, as demonstrated in [research outperforming](https://arxiv.org/abs/2605.12384v1) step-level approaches particularly for reasoning-intensive tasks. **Enhanced AI model provenance tracking through watermarking systems supports distillation protection**, as enabled by [TextSeal implementation](https://arxiv.org/abs/2605.12456v1) maintaining compatibility with serving optimizations. **Self-evolving red team frameworks enable adaptive security assessment of agent ecosystems**, as provided by [Proteus framework](https://arxiv.org/abs/2605.11891v1) revealing risks that single-shot evaluations miss. **Faster and more efficient safety moderation through specialized small models**, as demonstrated by [GLiGuard achieving](https://pioneer.ai/blog/gliguard-16x-faster-safety-moderation-with-a-small-language-model) 16x speedup over larger general-purpose models.
